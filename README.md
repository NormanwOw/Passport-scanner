## About
Desktop application for scan passport (RU) with UI, remote server and database

## Install
1. unzip `Tesseract-OCR.rar` into the root of project
2. `pip install -r requirements.txt`
3. `python server.py`
4. `python main.py`

or

1. run `Server.exe`
2. run `Scanner.exe`
